排序

std::sort

使用CPP内置STL库的 sort() 算法

用法:

// a[0] .. a[n - 1] 为需要排序的数列
// 对 a 原地排序，将其按从小到大的顺序排列
std::sort(a, a + n);

// cmp 为自定义的比较函数
std::sort(a, a + n, cmp);

std::sort(a + 1, a + 1 + n);                  // a[1] .. a[n] 从小到大排序
std::sort(a + 1, a + 1 + n, greater<int>());  // a[1] .. a[n] 从大到小排序
