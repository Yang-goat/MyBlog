# myBlog

一个基于 Markdown 的个人博客项目。

## 项目简介

myBlog 是一个简洁高效的博客系统，支持 Markdown 语法，便于快速发布和管理文章。

## 功能特性

- 支持 Markdown 编辑与渲染
- 文章分类与标签管理
- 文章搜索
- 响应式设计，适配多端
- 简单易用的界面

## 安装与使用

1. 克隆项目代码：

    ```bash
    git clone https://github.com/Yang-goat/MyBlog
    cd myBlog
    ```

2. 安装依赖：

    ```bash
    npm install
    ```

3. 启动项目：

    ```bash
    npm start
    ```

4. 访问本地服务：

    ```
    http://localhost:8080
    ```
