# linux 推荐编辑器

在 Linux 系统中，文本编辑器是最重要的工具之一。无论是修改配置文件、写脚本、查看日志，还是写代码，都离不开它。最常见、最实用的两个终端编辑器是：

- Vim：功能强大、学习曲线陡峭、效率极高
- Nano：简单直观、即学即用、适合新手

## 文件操作

### 新建空文件

```shell
touch 文件名
```

### 新建并写入内容

```shell
echo "hello world" > 文件名
```

### 新建+打开

```shell
vim 文件名
nano 文件名
```

## vim 入门指令

```
i       进入插入模式
Esc     回普通模式
:w      保存
:q      退出
:wq     保存退出
:q!     强制退出
dd      删除行
yy      复制行
p       粘贴
/xxx    搜索
:%s/a/b/g   替换
```

## nano入门

使用 nano 打开文件好哦后底部会有几乎所有的快捷键，无需额外学习。
