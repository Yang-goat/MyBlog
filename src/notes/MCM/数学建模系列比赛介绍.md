# 一文看懂数学建模系列比赛与备赛建议

::: tip 适合谁看
这篇文章适合准备参加 **数学建模比赛** 的本科生、研究生，以及想快速建立全年竞赛节奏感的同学。
:::

::: important 阅读建议
如果你是第一次接触数模，建议先看 **“总结与备赛路线”**；如果你已经准备参赛，可以直接按时间顺序查看全年赛事安排。
:::

## 全年赛事节奏总览

::: tabs

@tab 本科生推荐路线

**春季练手**（五一赛 / 数维杯）  
$\rightarrow$ **暑期冲刺**（国赛）  
$\rightarrow$ **寒假挑战**（美赛）

适合目标明确、希望逐步提升建模、编程、写作和团队协作能力的同学。

**春季适度热身**  
$\rightarrow$ **9月重点准备华为杯研赛**

:::

## 各赛事定位速览

| 类型 | 代表赛事 | 特点 | 适合人群 |
|:---:|:---|:---|:---|
| 国际顶级赛事 | [美赛（MCM/ICM）](https://www.comap.com/undergraduate/contests/) | 全球影响力最大，全英文写作，奖项含金量高 | 想冲国际奖项、英文能力较强的队伍 |
| 国内本科核心赛事 | [国赛（CUMCM）](http://www.mcm.edu.cn/) | 本科阶段最重要的数模赛事，认可度极高 | 本科生主力参赛队伍 |
| 研究生核心赛事 | [华为杯研赛](https://cpipc.acge.org.cn/) | 偏前沿科技与国家重大需求，题目更有科研味 | 研究生队伍 |
| 美赛模拟训练 | 华数杯、APMCM、小美赛、数维杯国际赛 | 赛制接近美赛，多为全英文写作 | 准备参加次年美赛的同学 |
| 国赛热身训练 | 妈妈杯、五一赛、数维杯春季赛 | 适合练完整流程，题风较接近国赛 | 初学者、本科备赛队伍 |
| 工科特色赛事 | 电工杯 | 偏电力、能源、工程应用 | 电气、自动化、能源相关专业学生 |

## 1月 - 2月：国际开年战

### 1.1 华数杯国际大学生数学建模竞赛

::: info 比赛信息
- **比赛时间：** 2026年1月17日 06:00 - 1月21日 09:00
- **报名时间：** 即日起至2026年1月16日 23:59
- **官网：** [赛氪网](https://www.saikr.com/vse/mcmicm2026)
- **报名费用：** 200元 / 队
:::

**赛事简介：**  
作为美赛前的“全真模拟”，赛制几乎完全对标美赛，比赛时长为 **96小时**，并且要求 **全英文写作**。赛题通常分为 **MCM（数学建模）** 和 **ICM（交叉学科）** 两类，非常适合在正式参加美赛前进行一次高强度适应训练。

**适合人群：**
- 计划参加美赛的队伍
- 想提前适应全英文论文写作的同学
- 想体验完整 96 小时比赛流程的初学者

---

### 1.2 美国大学生数学建模竞赛（MCM/ICM，美赛）

::: info 比赛信息
- **比赛时间：** 2026年1月30日 06:00 - 2月3日 09:00（北京时间）
- **报名时间：** 顾问注册于前一年10月开放，团队注册截止至2026年1月29日（美东时间）
- **官网：** [COMAP官网](https://www.comap.com/undergraduate/contests/)
- **报名费用：** 100美元 / 队（约合人民币700-800元）
:::

**赛事简介：**  
这是全球规模最大、影响力最广的国际性数学建模赛事之一，也是很多同学眼中的“数模天花板”。2026年新规中，论文页数上限调整为 **25页**，并且首次明确 **允许使用AI工具**，但要求在附录中说明使用情况。赛事奖项采用金字塔式分布，其中 **O奖（特等奖）** 比例极低，竞争非常激烈。

**关键词：**
- 国际认可度高
- 全英文论文
- 奖项含金量高
- 适合冲击高水平成果

::: details 参赛提醒
1. 美赛非常看重 **论文表达** 和 **故事完整性**。  
2. 英文写作能力往往和建模能力同样重要。  
3. 参赛前最好至少完成一次完整模拟。  
:::

## 3月 - 5月：春季练兵期

### 2.1 MathorCup 高校数学建模挑战赛（妈妈杯）

::: info 比赛信息
- **比赛时间：** 2026年4月17日 08:00 - 4月21日 09:00
- **报名时间：** 即日起至2026年4月16日 12:00
- **官网：** [赛氪网](https://www.saikr.com/vse/mathorcup2026)
- **报名费用：** 200元 / 队
:::

**赛事简介：**  
由中国优选法统筹法与经济数学研究会主办，赛题较多来自 **企业实际问题**，例如阿里、滴滴等背景题。整体上获奖比例相对友好，非常适合作为国赛前的第一次完整流程模拟。

**推荐理由：**
- 题目更贴近真实应用场景
- 适合练习论文结构和完整建模流程
- 对第一次参加校外比赛的同学比较友好

**适合人群：**
- 想先用轻量级赛事热身的同学
- 想尝试跨校组队的队伍
- 刚开始建立团队分工的初学者

**核心特点：**
- 题风贴近国赛
- 节奏适合练“国赛味”的论文
- 对备战 9 月国赛很有帮助

**更适合：**
- 电气、能源、自动化、控制类相关专业
- 对工程背景题更感兴趣的队伍
- 想用较低成本积累比赛经验的同学

## 7月 - 9月：国内巅峰赛

### 3.1 高教社杯全国大学生数学建模竞赛（CUMCM，国赛）

::: info 比赛信息
- **比赛时间：** 2026年9月10日 18:00 - 9月13日 20:00（预计）
- **报名时间：** 2026年5月1日 - 9月1日 20:00  
  > 通常由学校统一组织，校内选拔截止时间往往更早
- **官网：** [全国大学生数学建模竞赛官网](http://www.mcm.edu.cn/)
- **报名费用：** 300元 / 队（部分高校会为参赛队伍全额补贴）
:::

**赛事简介：**  
这是国内本科生层面最具代表性、含金量最高的数学建模赛事之一，由教育部高教司指导，中国工业与应用数学学会主办。比赛 **严禁跨校组队**，竞争激烈，国奖在 **保研、评优、综测、竞赛经历** 中通常有较高权重。

**为什么它最重要：**
- 本科阶段认可度极高
- 校内资源通常会向国赛倾斜
- 获奖难度大，但回报也高

::: warning 注意
国赛不只是“会做题”就够了，通常更考验：
- 团队分工是否成熟
- 论文排版是否稳定
- 48-72小时高压下的执行能力
- 对常见模型、算法、图表表达是否熟练
:::

**赛事简介：**  
这是研究生阶段最重要的数学建模竞赛之一，题目往往聚焦于 **前沿科技** 与 **国家重大需求**，如芯片、航天、碳中和、智能制造等方向。奖金设置通常也较为丰厚。

**适合人群：**
- 研究生队伍
- 有一定科研背景或工程经验的同学
- 希望把竞赛与未来研究方向结合起来的人
