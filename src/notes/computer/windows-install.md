# Windows 11 安装与重装教程

本文介绍如何从微软官网下载 Windows 11、制作启动 U 盘，并完成全新安装或系统重装。教程以支持 UEFI 的普通 x64 电脑为例。

::: danger 重要警告
全新安装会删除目标磁盘上的系统、软件和个人文件。执行删除分区或格式化操作前，必须确认数据已经备份，并根据磁盘容量和型号核对目标磁盘。
:::

## 一、安装前准备

### 1. 准备所需物品

- 一台可以正常联网的 Windows 电脑。
- 一个容量不小于 8 GB 的空 U 盘。
- 稳定的网络和电源。
- 需要安装系统的电脑。

制作启动盘会清空 U 盘，请先备份其中的文件。

### 2. 检查 Windows 11 硬件要求

Windows 11 的主要最低要求包括：

- 兼容的 64 位处理器，至少 2 核、1 GHz。
- 4 GB 内存。
- 64 GB 存储空间。
- UEFI 固件并支持安全启动。
- TPM 2.0。

旧电脑可以先运行微软的“电脑健康状况检查”工具。本文不介绍绕过 TPM、安全启动或处理器限制的方法，因为这可能导致更新和兼容性问题。

### 3. 备份数据和授权信息

至少备份以下内容：

- 桌面、下载、文档、图片和视频。
- 浏览器书签、密码和双重验证恢复码。
- 聊天软件、游戏存档和专业软件配置。
- SSH 密钥、开发环境配置和许可证文件。
- 网卡、存储控制器等关键驱动。

如果设备启用了 BitLocker，还应提前保存恢复密钥：

1. 访问 [Microsoft BitLocker 恢复密钥页面](https://aka.ms/myrecoverykey)。
2. 登录当前电脑使用的 Microsoft 账户。
3. 确认能够找到与本机对应的 48 位恢复密钥。
4. 将密钥额外保存在不会被本次安装清除的位置。

### 4. 确认系统版本

打开“设置 → 系统 → 系统信息”，记录当前版本是 Windows 11 家庭版还是专业版。重装时选择与数字许可证一致的版本，联网后通常可以自动激活。

## 二、制作 Windows 11 启动 U 盘

### 1. 下载官方工具

打开 [Windows 11 官方下载页面](https://www.microsoft.com/software-download/windows11)。

找到“创建 Windows 11 安装介质”，点击“立即下载”，获得 Media Creation Tool。不要从网盘或第三方系统站下载镜像。

### 2. 运行 Media Creation Tool

1. 插入准备好的 U 盘。
2. 右键运行下载的工具，必要时选择“以管理员身份运行”。
3. 接受许可条款。
4. 确认语言和 Windows 11 版本。
5. 在“选择要使用的介质”页面选择“U 盘”。
6. 选择正确的可移动磁盘。
7. 等待工具下载系统并写入 U 盘。

::: warning
选择 U 盘时应核对盘符和容量。选中的 U 盘会被格式化。
:::

完成后安全弹出 U 盘。为了避免安装时混淆，建议暂时拔掉无关的移动硬盘和 U 盘。

## 三、从 U 盘启动

### 1. 打开启动菜单

将启动 U 盘插入目标电脑并重新开机，在出现厂商 Logo 时连续按启动菜单快捷键。

常见按键如下，实际按键应以电脑或主板说明书为准：

| 品牌或主板 | 常见启动菜单按键 |
| --- | --- |
| Lenovo | F12 或 Fn + F12 |
| Dell | F12 |
| HP | F9 或先按 Esc |
| ASUS | Esc 或 F8 |
| Acer | F12 |
| MSI | F11 |
| Gigabyte | F12 |

### 2. 选择 UEFI 启动项

在启动菜单中选择名称类似下面的项目：

```text
UEFI: USB Flash Drive
```

如果同一个 U 盘同时出现 UEFI 和非 UEFI 两个选项，现代电脑应选择 UEFI 项。

如果看不到 U 盘：

1. 换一个 USB 接口后重试。
2. 重新制作启动盘。
3. 在 BIOS/UEFI 中确认 USB Boot 已启用。
4. 恢复固件默认设置后重新检查。

不要为了安装 Windows 11 随意关闭 TPM 或安全启动。

## 四、开始安装 Windows 11

### 1. 设置语言和键盘

进入安装界面后，选择语言、时间格式和键盘布局，然后点击“下一步”。

### 2. 选择安装 Windows 11

选择“安装 Windows 11”。如果安装器明确提示此次操作会删除文件、应用和设置，应在确认备份完成后再继续。

### 3. 处理产品密钥

如果电脑以前已经激活过相同版本的 Windows，可以选择“我没有产品密钥”，安装完成并联网后通常会自动激活。

新电脑或更换主板后的设备，应准备有效许可证。不要使用来源不明的激活工具。

### 4. 选择正确版本

选择与原许可证一致的版本：

- 原来是家庭版，选择 Windows 11 Home。
- 原来是专业版，选择 Windows 11 Pro。

版本不一致可能导致无法自动激活。

## 五、磁盘分区与安装位置

这是整个流程中风险最高的步骤。安装器会显示“磁盘 0”“磁盘 1”以及各磁盘上的分区。

### 方案 A：整盘全新安装（推荐用于彻底重装）

适用于目标磁盘上的所有数据都可以删除，并且 Windows 将独占该磁盘的情况。

1. 根据总容量确认目标磁盘，例如 476 GB 通常对应标称 512 GB 的 SSD。
2. 逐个删除目标磁盘上的系统、MSR、恢复和主分区。
3. 删除后，目标磁盘应只剩“未分配空间”。
4. 选中该未分配空间，点击“下一步”。

Windows 会自动建立 EFI、MSR、系统分区和恢复分区，不需要手工创建。

::: danger
多硬盘电脑不能机械照搬“删除磁盘 0”。磁盘编号可能发生变化，必须结合容量、型号和现有分区判断。无法确认时应关机并暂时断开非目标硬盘。
:::

### 方案 B：仅格式化原 Windows 分区

适用于同一磁盘还有需要保留的数据分区，并且你能够准确识别原系统分区的情况。

1. 找到原 Windows 主分区，通常容量较大。
2. 不要删除 EFI、恢复分区或需要保留的数据分区。
3. 选中原 Windows 分区并点击“格式化”。
4. 选中格式化后的分区继续安装。

这种方式可能保留旧的引导和恢复分区。若磁盘结构已经混乱，备份数据后使用整盘全新安装更稳妥。

### 是否需要提前划分 C 盘和 D 盘

单硬盘电脑可以直接让 Windows 使用整盘，安装完成后再通过“磁盘管理”压缩 C 盘并创建数据分区。

若希望安装时划分：

1. 选中未分配空间并点击“新建”。
2. 输入系统分区容量，单位为 MB。
3. 例如计划分配约 200 GB，可输入 `204800`。
4. Windows 会额外创建必要的小型系统分区。
5. 选中主要系统分区安装 Windows。
6. 剩余未分配空间可在安装完成后创建为 D 盘。

不建议给 C 盘分配过小空间。日常开发或游戏电脑通常应预留至少 150 至 200 GB，并根据软件规模增加。

## 六、等待安装完成

点击“下一步”后，安装器会复制文件并多次重启。

第一次重启后：

- 可以拔出启动 U 盘；或
- 不再按启动菜单按键，让电脑从内部硬盘启动。

如果再次回到安装首页，说明电脑又从 U 盘启动了。拔出 U 盘后重新启动即可。

## 七、完成首次开机设置

按界面提示完成：

1. 选择国家或地区。
2. 选择键盘布局。
3. 连接网络。
4. 登录 Microsoft 账户。
5. 设置 PIN。
6. 根据需要调整隐私和同步选项。

Windows 11 家庭版以及个人使用的专业版通常要求联网并登录 Microsoft 账户完成首次设置。

## 八、安装后的必要检查

### 1. 运行 Windows Update

打开“设置 → Windows 更新”，重复检查更新并重启，直到没有重要更新。

### 2. 检查设备驱动

打开“设备管理器”，确认没有黄色感叹号。优先从电脑或主板厂商官网下载：

- 芯片组驱动。
- 无线网卡和有线网卡驱动。
- 显卡驱动。
- 声卡、触控板和快捷键驱动。

### 3. 检查激活状态

打开“设置 → 系统 → 激活”，确认系统显示已激活。

### 4. 创建数据分区

如果安装时保留了未分配空间：

1. 右键开始菜单，打开“磁盘管理”。
2. 找到目标磁盘上的未分配空间。
3. 右键选择“新建简单卷”。
4. 分配盘符并使用 NTFS 格式化。

### 5. 恢复文件和软件

先安装安全更新和驱动，再恢复个人文件、开发环境和常用软件。确认系统稳定后再清理备份。

## 常见问题

### 安装器找不到 SSD

常见原因是 Intel VMD、RST 或 RAID 模式需要存储控制器驱动。优先查阅电脑厂商的安装说明并下载对应驱动。不要在不清楚影响的情况下直接修改 SATA/VMD 设置，否则旧系统可能无法启动。

### 安装后没有网络

使用另一台电脑从设备厂商官网下载网卡驱动，复制到 U 盘后安装。

### 提示无法安装到该磁盘

现代 Windows 11 通常应使用 UEFI + GPT。如果启动盘以旧式 Legacy/CSM 模式启动，可能与磁盘分区表不匹配。返回启动菜单并选择带 `UEFI` 的 U 盘项目。

### 只想修复系统，不想删除文件

如果原 Windows 仍能启动，可以在系统内打开安装 U 盘中的 `setup.exe`，选择保留个人文件和应用进行就地重装。也可以先查看“设置 → 系统 → 恢复”中的恢复选项。
