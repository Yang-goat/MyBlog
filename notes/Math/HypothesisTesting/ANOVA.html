方差分析

一、什么是方差分析？

方差分析（Analysis of Variance, ANOVA）用于检验多个总体均值是否相等。

其核心思想是比较：

  • 组间变异（不同组均值之间的差异，解释为“因素效应”）
  • 组内变异（组内个体间的随机差异，即“误差”）

记第 iii 组第 jjj 个观测值为 YijY_{ij}Yij​，方差分析要判断：

不同组的平均值  μi \,\mu_i\,μi​（总体均值）是否都相同？

二、为什么用方差分析？

当组数 ≥ 3，若用多次 t 检验，会导致：

  • 错误率显著上升
  • 推断不再可靠

ANOVA 提供“一次整体检验”来判断：

H0: μ1=μ2=⋯=μkH_0:\ \mu_1 = \mu_2 = \cdots = \mu_kH0​: μ1​=μ2​=⋯=μk​

是否成立。

其中 kkk 表示组数（因素水平数）。

三、一元方差分析（One-way ANOVA）模型

模型写为：

Yij=μ+αi+εijY_{ij} = \mu + \alpha_i + \varepsilon_{ij} Yij​=μ+αi​+εij​

其中：

  • μ\muμ：总体均值（基准平均水平）
  • αi\alpha_iαi​：因素第 iii 水平的效应（组均值相对 μ\muμ 的偏移）
  • εij\varepsilon_{ij}εij​：随机误差项（独立、均值 0、方差 σ2\sigma^2σ2）

第 iii 组样本均值记为 Xˉi\bar X_iXˉi​，所有样本的总体平均记为 Xˉ\bar XXˉ。

四、方差分解（ANOVA 核心思想）

总变异可以拆分为：

SST=SSB+SSWSS_T = SS_B + SS_W SST​=SSB​+SSW​

其中：

  • SSTSS_TSST​（Total Sum of Squares）总平方和
    衡量所有数据偏离总体均值 Xˉ\bar XXˉ 的程度。

  • SSBSS_BSSB​（Between Groups）组间平方和
    衡量各组均值 Xˉi\bar X_iXˉi​ 相对总体均值的差异，用来度量“因素效应”。

    SSB=∑i=1kni(Xˉi−Xˉ)2SS_B = \sum_{i=1}^k n_i(\bar X_i - \bar X)^2 SSB​=i=1∑k​ni​(Xˉi​−Xˉ)2

    其中 nin_ini​ 为第 iii 组的样本量。

  • SSWSS_WSSW​（Within Groups）组内平方和
    衡量每组内的随机误差：

    SSW=∑i=1k∑j=1ni(Yij−Xˉi)2SS_W = \sum_{i=1}^k\sum_{j=1}^{n_i}(Y_{ij}-\bar X_i)^2 SSW​=i=1∑k​j=1∑ni​​(Yij​−Xˉi​)2

五、自由度（df）与均方（MS）

自由度：

  • 组间自由度：dfB=k−1df_B = k - 1dfB​=k−1
  • 组内自由度：dfW=N−kdf_W = N - kdfW​=N−k
  • 总自由度：dfT=N−1df_T = N - 1dfT​=N−1

其中 NNN 为总样本量，N=∑niN = \sum n_iN=∑ni​，kkk为组数。

均方（Mean Square）是平方和除以自由度：

MSB=SSBdfB,MSW=SSWdfWMS_B = \frac{SS_B}{df_B},\qquad MS_W = \frac{SS_W}{df_W} MSB​=dfB​SSB​​,MSW​=dfW​SSW​​

六、F 统计量与 p 值

F 值定义为：

F=MSBMSWF = \frac{MS_B}{MS_W} F=MSW​MSB​​

含义：

  • 若组间变异 (MSBMS_BMSB​) 远大于组内变异 (MSWMS_WMSW​)
    → FFF 大
    → 说明因素效应显著。

p 值定义为在 H0H_0H0​（均值相等）成立下：

p=P(F(dfB,dfW)≥Fobs)p = P(F_{(df_B,df_W)} \ge F_{\text{obs}}) p=P(F(dfB​,dfW​)​≥Fobs​)

若 p<αp < \alphap<α（通常 α=0.05\alpha=0.05α=0.05）则拒绝 H0H_0H0​。

七、单因素方差分析流程总结

  1. 计算组均值 Xˉi\bar X_iXˉi​ 与总体均值 Xˉ\bar XXˉ
  2. 计算 SSB,SSW,SSTSS_B, SS_W, SS_TSSB​,SSW​,SST​
  3. 分配自由度：k−1k-1k−1, N−kN-kN−k, N−1N-1N−1
  4. 计算均方 MSBMS_BMSB​, MSWMS_WMSW​
  5. 计算 F 值
  6. 计算 p 值
  7. 判断是否拒绝原假设

九、双因素、三因素方差分析（扩展简介）

1. 双因素方差分析（Two-way ANOVA）

当存在两个因素（如肥料 A、灌溉 B）时，模型可检验：

  • A 的主效应
  • B 的主效应
  • A×B 的交互效应（因素之间是否互相影响）

方差分解：

SST=SSA+SSB+SSAB+SSESS_T = SS_A + SS_B + SS_{AB} + SS_E SST​=SSA​+SSB​+SSAB​+SSE​

其中：

  • SSASS_ASSA​：因素 A 对均值差异的解释部分
  • SSBSS_BSSB​：因素 B 的效应
  • SSABSS_{AB}SSAB​：交互作用的效应
  • SSESS_ESSE​：误差平方和

2. 三因素方差分析（Three-way ANOVA）

当有三个因素（如温度 A、压力 B、催化剂 C）时，可检验：

  • A、B、C 三个主效应
  • 三个二阶交互：A×B、A×C、B×C
  • 一个三阶交互：A×B×C

方差分解：

SST=SSA+SSB+SSC+SSAB+SSAC+SSBC+SSABC+SSESS_T = SS_A + SS_B + SS_C + SS_{AB} + SS_{AC} + SS_{BC} + SS_{ABC} + SS_E SST​=SSA​+SSB​+SSC​+SSAB​+SSAC​+SSBC​+SSABC​+SSE​

每一部分都有自己的自由度、均方和 F 检验。

十、总结

方差分析核心逻辑：

  1. 将总方差拆分为组间（因素效应）+ 组内（误差）
  2. 计算 F=MSB/MSWF = MS_B / MS_WF=MSB​/MSW​，比较信号与噪声
  3. p 值来自 F 分布右尾概率
  4. 显著则拒绝均值相等的原假设

扩展形式（双因素、三因素）仅是在同一框架下增加主效应与交互效应的分解。
